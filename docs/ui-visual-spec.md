# Aurora 视觉法则（单一设计之手）

状态：v1（2026-07-15）。目的：消除"东拼西凑"。所有页面必须共用同一套组件与节奏，禁止各页自造样式。新增/改页面一律遵守本文件；页面只允许用 widgets/ 里的既有组件拼装，不得在页面文件里写死颜色、圆角、阴影、间距。

## 一、基调
- 纯白不透明基底；蓝→粉斜渐变仅作强调色，只出现在：主 CTA 按钮填充、进度条、选中/焦点、导航选中图标。其余一律不上彩色。
- 干净、通透、克制、大留白。宁可少不可乱。禁止装饰性色块/边框堆叠。

## 二、布局
- 左侧固定图标导航栏（64px，不展开），选中项为白色浮起圆角胶囊（麦麦式）。
- 内容区外边距统一 32px（SPACE_XL）。
- 每页结构固定：页头（标题+副标题）-> 内容分区。分区之间 24px（SPACE_LG）。

## 三、页头（所有页面统一）
- 标题：TEXT_TITLE(24) / 粗 / on_surface。
- 副标题：TEXT_BODY(14) / on_surface_muted，位于标题下 4px。
- 页头与下方内容间距 24px。用统一组件 widgets 的 page_header(title, subtitle)。

## 四、卡片（唯一卡片样式，禁止变体）
- 背景纯白；圆角 RADIUS_MD(14)；内边距 20px；描边 1px（中性极淡，如 rgba(dark,0.08)）；柔和阴影（offset(0,2) blur 12，shadow 令牌）。
- 卡片之间间距 16px（SPACE_MD）。卡内分区标题用 TEXT_HEADING(17)。

## 五、按钮（两种，固定）
- 主 CTA：蓝→粉渐变填充、白字、圆角 RADIUS_SM(8)、内边距 [10,16]、按压弹簧(aurora_press)。全页至多一个主 CTA。
- 次按钮：白底 + 极淡描边 + 深色字，同圆角内边距。
- 禁止第三种按钮样式。

## 六、图标与文字
- 图标：细线描边。默认 on_surface_muted；选中/激活态才用强调色。尺寸 16 或 20。
- 文字层级只用刻度：标题24 / 分区17 / 正文14 / 说明12。颜色只用 on_surface 与 on_surface_muted。

## 七、空态（统一组件）
- 居中：一枚 line 图标（muted）+ 一行说明（on_surface_muted）+（可选）一个次按钮。用 widgets 的 empty_state。

## 八、动效
- 弹簧默认 anim::aurora()（bounce0.35/dur0.30）；页面入场轻微 stagger 上滑；按压 aurora_press；导航固定不再有展开动画。所有帧走 window::frames()。

## 九、执行纪律
- 页面文件禁止出现字面色值/圆角/阴影/魔数间距——一律引用 theme 刻度与 widgets 组件。
- 加新页 = 复用 page_header + card + button + empty_state + section 组合，天然与其它页一致。
- 本法则先在主页落地为标杆，确认后原样套到其它页。
